Homebrew repository to install tools from us on your macOS.


## Prepare

```bash
brew tap webhippie/webhippie
```


## Install

### [errors](https://github.com/webhippie/errors)

```bash
brew install errors
errors -h
```

### [ldap-proxy](https://github.com/webhippie/ldap-proxy)

```bash
brew install ldap-proxy
ldap-proxy -h
```
